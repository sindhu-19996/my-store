# 🛍 MyStore - E-commerce App

## 🚀 Features
- Product listing with search
- Product detail page
- Add to cart functionality
- Guest & user login
- Responsive design
- SEO optimized pages

## 🛠 Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## 📦 Setup

npm run dev
